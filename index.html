JSX → HTML Converter

Paste or upload any Claude artifact (.jsx/.tsx). Imports are auto-resolved live from esm.sh, so any npm package works — not just a fixed list. Runs entirely in your browser.

Live preview
How this works: every import in your file (e.g. recharts, lucide-react, framer-motion) is automatically mapped to esm.sh, which serves that package as a real browser module on demand — so this isn't limited to a fixed library list. React/ReactDOM are pinned to a single shared instance so hooks work correctly across libraries. Requires internet access when the output HTML is opened, since packages are fetched live rather than bundled offline.
