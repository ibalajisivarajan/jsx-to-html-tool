JSX → HTML Converter

Paste or upload a Claude artifact (.jsx/.tsx) and download a standalone HTML file. Runs entirely in your browser — nothing is uploaded anywhere.

Live preview
Note on library support: This runs entirely client-side using Babel Standalone, no bundler. It includes React, ReactDOM, Tailwind, Recharts, and Framer Motion via CDN. Components using only these plus standard React hooks will work reliably. Artifacts with other npm imports (e.g. less common icon sets) may need those swapped for inline SVG or a CDN equivalent.
